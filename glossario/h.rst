.. _H:

H
==

.. glossary::

    HTML
        Abreviação para **HyperText Markup Language**, ou *Linguagem para Marcação de HiperTexto*, 
        em Português. A Linguagem é composta de marcações, também conhecidas como :term:`tags`, 
        que informam como o texto será formatado.
    
    HTTP
        Abreviação para **HyperText Transfer Protocol**, ou *Protocolo para Transferência de HiperTexto*, 
        em Português.
